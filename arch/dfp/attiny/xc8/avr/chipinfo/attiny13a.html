ATTINY13A Support Information

XC8-CC #pragma config Usage

#pragma config <setting>=<named value>
For example:
// Select Clock Source : Int. RC Osc. 9.6 MHz; Start-up time: 14 CK + 64 ms
// Divide clock by 8 internally : SET
// Watch-dog Timer always on : CLEAR
// Preserve EEPROM through the Chip Erase cycle : CLEAR
// Serial program downloading (SPI) enabled : SET
#pragma config SUT_CKSEL = SUT_CKSEL_INTRCOSC_9MHZ6_14CK_64MS, CKDIV8 = SET, WDTON = CLEAR, EESAVE = CLEAR, SPIEN = SET
#pragma config <setting>=<literal constant>
For example:
// Select Clock Source : Int. RC Osc. 9.6 MHz; Start-up time: 14 CK + 64 ms
// Divide clock by 8 internally : SET
// Watch-dog Timer always on : CLEAR
// Preserve EEPROM through the Chip Erase cycle : CLEAR
// Serial program downloading (SPI) enabled : SET
#pragma config SUT_CKSEL = 0xA, CKDIV8 = 0x0, WDTON = 0x1, EESAVE = 0x1, SPIEN = 0x0
