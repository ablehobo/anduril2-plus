ATTINY26 Support Information

XC8-CC #pragma config Usage

#pragma config <setting>=<named value>
For example:
// Select Clock Source : Int. RC Osc. 1 MHz; Start-up time: 6 CK + 64 ms
// CKOPT fuse (operation dependent of CKSEL fuses) : CLEAR
#pragma config PLLCK_SUT_CKSEL = PLLCK_SUT_CKSEL_INTRCOSC_1MHZ_6CK_64MS, CKOPT = CLEAR
#pragma config <setting>=<literal constant>
For example:
// Select Clock Source : Int. RC Osc. 1 MHz; Start-up time: 6 CK + 64 ms
// CKOPT fuse (operation dependent of CKSEL fuses) : CLEAR
#pragma config PLLCK_SUT_CKSEL = 0xA1, CKOPT = 0x1
