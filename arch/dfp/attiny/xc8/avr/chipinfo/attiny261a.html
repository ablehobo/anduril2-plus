ATTINY261A Support Information

XC8-CC #pragma config Usage

#pragma config <setting>=<named value>
For example:
// Select Clock source : Int. RC Osc. 8 MHz; Start-up time PWRDWN/RESET: 6 CK/14 CK + 64 ms
// Clock output on PORTB5 : CLEAR
// Divide clock by 8 internally : SET
#pragma config SUT_CKSEL = SUT_CKSEL_INTRCOSC_8MHZ_6CK_14CK_64MS, CKOUT = CLEAR, CKDIV8 = SET
#pragma config <setting>=<literal constant>
For example:
// Select Clock source : Int. RC Osc. 8 MHz; Start-up time PWRDWN/RESET: 6 CK/14 CK + 64 ms
// Clock output on PORTB5 : CLEAR
// Divide clock by 8 internally : SET
#pragma config SUT_CKSEL = 0x22, CKOUT = 0x1, CKDIV8 = 0x0
