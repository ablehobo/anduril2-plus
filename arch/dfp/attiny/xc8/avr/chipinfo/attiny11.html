ATTINY11 Support Information

XC8-CC #pragma config Usage

#pragma config <setting>=<named value>
For example:
// Clock Select : Internal RC Oscillator
// External reset function of PB5 disabled : SET
// Short start-up time selected : CLEAR
#pragma config CKSEL = CKSEL_INTRCOSC, RSTDISBL = SET, FSTRT = CLEAR
#pragma config <setting>=<literal constant>
For example:
// Clock Select : Internal RC Oscillator
// External reset function of PB5 disabled : SET
// Short start-up time selected : CLEAR
#pragma config CKSEL = 0x4, RSTDISBL = 0x0, FSTRT = 0x1
