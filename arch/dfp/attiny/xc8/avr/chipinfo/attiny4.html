ATTINY4 Support Information

XC8-CC #pragma config Usage

#pragma config <setting>=<named value>
For example:
// Disable external reset : CLEAR
// Watch dog timer always on : CLEAR
// Output external clock : CLEAR
#pragma config RSTDISBL = CLEAR, WDTON = CLEAR, CKOUT = CLEAR
#pragma config <setting>=<literal constant>
For example:
// Disable external reset : CLEAR
// Watch dog timer always on : CLEAR
// Output external clock : CLEAR
#pragma config RSTDISBL = 0x1, WDTON = 0x1, CKOUT = 0x1
