ATTINY828 Support Information

XC8-CC #pragma config Usage

#pragma config <setting>=<named value>
For example:
// Select Clock Source : Int. RC Osc. 8 MHz; Start-up time PWRDWN/RESET: 6 + 14 CK + 65 ms
// Clock output on PORTB0 : CLEAR
// Divide clock by 8 internally : SET
#pragma config SUT_CKSEL = SUT_CKSEL_INTRCOSC_8MHZ_6CK_14CK_65MS, CKOUT = CLEAR, CKDIV8 = SET
#pragma config <setting>=<literal constant>
For example:
// Select Clock Source : Int. RC Osc. 8 MHz; Start-up time PWRDWN/RESET: 6 + 14 CK + 65 ms
// Clock output on PORTB0 : CLEAR
// Divide clock by 8 internally : SET
#pragma config SUT_CKSEL = 0x22, CKOUT = 0x1, CKDIV8 = 0x0
