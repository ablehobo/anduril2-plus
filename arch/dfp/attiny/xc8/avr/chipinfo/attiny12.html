ATTINY12 Support Information

XC8-CC #pragma config Usage

#pragma config <setting>=<named value>
For example:
// Clock Select : Internal RC Ocsillator
// External reset function of PB5 disabled : CLEAR
// Serial program downloading (SPI) enabled : SET
// Brown-out detection enabled : CLEAR
// Brown-out Detection Level : Brown-out detection at VCC=2.7 V
#pragma config CKSEL = CKSEL_INTRCOSC, RSTDISBL = CLEAR, SPIEN = SET, BODEN = CLEAR, BODLEVEL = BODLEVEL_2V7
#pragma config <setting>=<literal constant>
For example:
// Clock Select : Internal RC Ocsillator
// External reset function of PB5 disabled : CLEAR
// Serial program downloading (SPI) enabled : SET
// Brown-out detection enabled : CLEAR
// Brown-out Detection Level : Brown-out detection at VCC=2.7 V
#pragma config CKSEL = 0x2, RSTDISBL = 0x1, SPIEN = 0x0, BODEN = 0x1, BODLEVEL = 0x0
