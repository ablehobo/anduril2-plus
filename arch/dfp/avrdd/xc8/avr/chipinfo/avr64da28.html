AVR64DA28 Support Information

XC8-CC #pragma config Usage

#pragma config <setting>=<named value>
For example:
// Watchdog Timeout Period : Watch-Dog timer Off
// Watchdog Window Timeout Period : Window mode off
#pragma config PERIOD = PERIOD_OFF, WINDOW = WINDOW_OFF
#pragma config <setting>=<literal constant>
For example:
// Watchdog Timeout Period : Watch-Dog timer Off
// Watchdog Window Timeout Period : Window mode off
#pragma config PERIOD = 0x0, WINDOW = 0x0

Register: BODCFG
SLEEP BOD Operation in Sleep Mode
SLEEP_DISABLE BOD disabled
SLEEP_ENABLE BOD enabled in continuous mode
SLEEP_SAMPLE BOD enabled in sampled mode
ACTIVE BOD Operation in Active Mode
ACTIVE_DISABLE BOD disabled
ACTIVE_ENABLE BOD enabled in continuous mode
ACTIVE_SAMPLE BOD enabled in sampled mode
ACTIVE_ENABLEWAIT BOD enabled in continuous mode. Execution is halted at wake-up until BOD is running.
